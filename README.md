# A fastify boilerplate with typescript

## usage

### bootstrap

- Docker mode

```bash
sh start.sh
```

- Hot reload mode

```bash
pnpm dev
```

- Preview mode

```bash
pnpm build
# start server
pnpm runner:local
```
